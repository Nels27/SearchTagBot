An ECE 4180 Final Design Project

Find Out More

General Overview

This project is a culmination of features aspects of mapping, robotics, using api on cloud computing, a GUI, and computer vision. Its main objective is to be teleoperated in an environment that contains army men in various areas and identify them and tag them. Then it would provide data to a GUI that would recreate that same map and show areas where the army men were tagged and where they were not tagged.
